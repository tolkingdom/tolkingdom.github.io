VM Configuration

VMware Workstation Player

Vmware Workstation Player is a free type-two hypervisor created by VMware that allows you to configure and run virtual machines on directly on top of your local computers pre-existing operating system. This is perfect for the simple home-lab environment. The main limitation with the free version of this software is that it only allows you to run one VM at a time. This won’t be an issue in our scenario, so we will be using this software throughout this guide.

  • VMware Workstation Player Download Page

If you are looking to run more than one VM at a time and still need a free type-two hypervisor, we reccomend VirtualBox
